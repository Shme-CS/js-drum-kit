# Drum Kit Sound Files

This directory contains the audio files for the drum kit.

## Required Sound Files

Place the following MP3 files in this directory:

- `kick.mp3` - Kick drum sound
- `snare.mp3` - Snare drum sound
- `hihat.mp3` - Hi-hat sound
- `tom.mp3` - Tom drum sound
- `clap.mp3` - Clap sound
- `ride.mp3` - Ride cymbal sound

## Where to Get Free Drum Sounds

You can download free drum samples from:

1. **Freesound.org** - https://freesound.org/
   - Search for "kick drum", "snare", "hi-hat", etc.
   - Download as MP3 or WAV format

2. **SampleSwap** - https://sampleswap.org/
   - Free drum samples in various formats

3. **99Sounds** - https://99sounds.org/
   - Free sample packs

4. **Bedroom Producers Blog** - https://bedroomproducersblog.com/
   - Free sample libraries

## File Format

- Format: MP3 or WAV
- Recommended: Short samples (< 2 seconds)
- Quality: 44.1kHz, 16-bit minimum
