# Screenshots

This directory contains project screenshots for documentation.

## Required Screenshots

- `desktop.png` - Desktop view (1920x1080)
- `tablet.png` - Tablet view (768x1024)
- `mobile.png` - Mobile view (375x667)
- `demo.gif` - Animated demo

## How to Add Screenshots

1. Take screenshots of the application
2. Optimize images for web (compress)
3. Name them according to the list above
4. Place them in this directory
