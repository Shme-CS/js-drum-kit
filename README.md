# JavaScript Drum Kit

A interactive drum kit built with vanilla JavaScript.

## Project Structure

```
js-drum-kit/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── README.md
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start playing the drums!

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
